<Card.Body>
Podemos cambiar el color de un `favicon` dependiendo del `color-schema` de nuestro dispositivo.

Para hacer esto necesitamos agregar dentro de nuestra etiqueta svg lo sigiuente:

```html
<svg>
  <style>
    path {
      fill: light-dark(var(--dark-transparent), var(--white-transparent));
    }
  </style>
</svg>
```
