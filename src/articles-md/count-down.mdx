Hacer un count down no parece ser tan dificil, aunque si atemorizar al comienzo.

Para hacer esto con JavaScript puedes ver este codigo:

```js
const date = 1716231600000

const SECOND = 1000
const MINUTE = SECOND * 60
const HOUR = MINUTE * 60
const DAY = HOUR * 24

const init = () => {
  const formatDate = (date: number) => {
    return Math.floor(date).toString().padStart(2, '0')
  }

  const updateCountDown = () => {
    const now = Date.now()
    const diff = date - now

    const diffDays = formatDate(diff / DAY)
    const diffHours = formatDate((diff % DAY) / HOUR)
    const diffMinutes = formatDate((diff % HOUR) / MINUTE)
    const diffSecons = formatDate((diff % MINUTE) / SECOND)

    console.log(
      `D:${diffDays}, H:${diffHours}, M:${diffMinutes}, S:${diffSecons} seconds`
    )
  }

  setInterval(updateCountDown, SECOND)
}

init()
```

> Dato adicional: Para evitar que el contador se desface por el ancho de cada caracter numerico, podemos usar la propiedad `font-variant-numeric: tabular-nums;` en CSS que va a hacer que cada caracter tenga el mismo ancho. De esa manera un `1` va a ocular lo mismo que un `0`, por mas que el `1` sea mas angosto.
